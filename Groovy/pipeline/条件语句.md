## 条件语句

### if语句

```groovy
String branch = "prod"
if ( branch == "dev" ){
    println("dev....")
} else if ( branch == "test" ){
    println("test....")
} else {
    println("prod....")
}
```

switch语句

```groovy
String branch = "prod"
switch(branch) {
	case "dev":
		println("dev....")
		break
	case "test":
		println("test....")
		break
	default:
		println("prod....")
}
```

for循环

```groovy
// 遍历0-9，打印
for ( i=1;i<10;i++ ){
	println(i)
}

10.times { i ->
	println(i)
}

// 遍历List
def nameList = ["zhangsan", "lisi", "wangwu"]
for ( i in nameList ){
    println(i)
}

// 使用each遍历map
def stus = ["jiangsu":"nanjing", "anhui":"hefei"]
stus.each { k, v ->
    println(k + "=" + v)
}

for ( k in stus.keySet()){
    println(k+"="+stus[k])
}
```
